
### 使用说明（用户手册）

#### 命令总览与格式

| 命令            | 说明                           | 示例           |
| --------------- | ------------------------------ | -------------- |
| `new`           | 创建新文件系统（清空所有数据） | new            |
| `sfs <file>`    | 加载已保存的文件系统           | sfs myfs.dat   |
| `exit <file>`   | 保存当前文件系统并退出         | exit myfs.dat  |
| `mkdir <name>`  | 创建子目录                     | mkdir docs     |
| `rmdir <name>`  | 删除空子目录                   | rmdir docs     |
| `ls`            | 显示当前目录下所有项           | ls             |
| `cd <dir>`      | 进入子目录                     | cd media       |
| `cd ..`         | 返回上一级目录                 | cd ..          |
| `create <file>` | 创建新文件                     | create log.txt |
| `open <file>`   | 打开文件进行写入/读取          | open log.txt   |
| `write <text>`  | 向当前打开的文件追加文本       | write Hello    |
| `read`          | 读取当前打开文件内容           | read           |
| `close`         | 关闭当前打开文件               | close          |
| `delete <file>` | 删除当前目录下文件             | delete log.txt |

#### 注意事项

* `open` 后只能写入/读取该文件，需 `close` 后再操作其他文件。
* `rmdir` 仅能删除空目录。
* 不支持重名文件或目录。
* `exit` 保存到文件（如 `myfs.dat`），下次用 `sfs` 加载。
* 所有内容保存在内存中，未 `exit` 前关闭程序会丢失数据。
